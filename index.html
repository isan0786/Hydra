Hydra Installation On-Premise/Local

Prequisites

  1. Docker installation - Docker
  2. Docker Compose installation - Docker compose installation guide
  3. Git - Git

Installation Steps

  1. Go to Github Hydra
  2. Clone the project under "Quick Start" with command -> git clone https://github.com/hydradatabase/hydra && cd hydra
  3. Edit the docker-compose.yml file, and add version in the beginning of the file based on the installation version. For eg - version:"2.2" for more information on How to write docker compose files
  4. Execute rest of the commands
  5. At this point you should have "Hydra" container up, running and connected! Hurray!!!

Create Database & Table using Command line

Run the below Postgres queries in sequence but before that make sure to be connected with hydra database. Using the command psql postgres://postgres:hydra@127.0.0.1:5484

  1. CREATE DATABASE mydatabase;
  2. \c mydatabase;
  3. CREATE TABLE mytable ( id SERIAL PRIMARY KEY, name VARCHAR(50), age INTEGER, email VARCHAR(100) );
  4. INSERT INTO mytable (name, age, email) VALUES ('John Smith', 35, 'john.smith@example.com'), ('Jane Doe', 28, 'jane.doe@example.com'), ('Bob Johnson', 42, 'bob.johnson@example.com'), ('Emily Davis', 21, 'emily.davis@example.com'), ('Mike Wilson', 39, 'mike.wilson@example.com');
  5. SELECT * FROM mytable;

    6. At this point you are expert enough to execute any "SQL CRUD operations" on Hydra. Still feel rusty!?!? then go to CRUD Commands Tutorials
